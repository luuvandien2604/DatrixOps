# Audit Log

Open **Manage → Audit** to review recent workspace activity in reverse chronological order.

Each event records the actor, action, affected resource, timestamp, and available request context. Use the audit log to investigate unexpected changes, validate administrative work, and coordinate safely in multi-user environments.
