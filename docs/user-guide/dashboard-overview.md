# Dashboard Overview

The application is organized into three areas:

1. **Sidebar:** Primary navigation for overview, servers, metrics, alerts, websites, and administration.
2. **Top bar:** Search, theme controls, notifications, and account access.
3. **Workspace:** The current operational view.

## Overview

The overview summarizes connected servers, online health, CPU and memory telemetry, recent agent activity, and firing incidents. Values come from agent heartbeats and refresh automatically.

## Monitoring

Monitoring displays real time-series data. Missing heartbeats remain empty instead of being interpolated, making offline periods visible on the timeline.
