# Website & SSL Monitoring

## Add a website

1. Open **Websites**.
2. Select **Add Website**.
3. Enter a display name and complete URL, including `https://`.
4. Save the website.

DatrixOps checks reachability and reads the SSL certificate issuer and expiration date. Each card shows current status, remaining certificate lifetime, and the latest check.

Deleting a website only stops monitoring; it does not modify the external website.
