# Platform Capabilities

- **Hardware telemetry:** CPU, memory, disks, network throughput, load, and process visibility.
- **Server organization:** Groups and tags for search, filtering, and policy assignment.
- **Alerting:** CPU, memory, and offline detection with Telegram and Discord delivery.
- **Workspace isolation:** User-scoped infrastructure and role-based access.
- **Audit history:** Trace important configuration and resource changes.
- **API access:** Revocable keys for automation and third-party integrations.
- **Website monitoring:** HTTP availability and SSL certificate lifetime.
- **Remote operations:** Agent-mediated Docker actions and update tasks.
- **Agent lifecycle:** Centralized installation and safe upgrade workflows.
